fadp平台后端目录结构介绍


  framework                                     平台基础目录（严禁修改）
  hot-deploy                                    应用目录
      demo                                      演示模块
          config                                应用配置（放properties文件等，较少使用）
          data                                  存放种子数据及演示数据
            SeedData.xml                        种子数据文件
          entitydef                             实体定义目录
            demo-entitymodel.xml                实体定义文件
          lib                                   jar包目录（平台提供的jar包已经能满足多数情况，较少使用）
          script                                脚本目录（主要存放groovy脚本，用于处理简单业务）
          src                                   java代码目录
          webapp                                web目录
              demo                              具体web应用目录
                WEB-INF
                  controller.xml                web请求控制器配置文件
                  web.xml                       与普通web项目的web.xml类似
          build.xml                             ant构建配置文件
          fadp-component.xml
      entityMgr                                 平台提供的数据库管理工具
      fadp                                      平台提供的基础模块
  runtime                                       平台数据存放位置（其中包含内嵌数据库H2的数据库文件）
