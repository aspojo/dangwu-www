通用树组件 app-common-type-tree

用于将一个数据库表中的数据以树状形式展示，并提供树节点选中事件监听功能。使用该组件的表需要符合一个树状结构数据表的特征：即将树数据以节点为单位存放，每个节点记录其父节点。

使用标签app-common-type-tree
导入模块CommonTypeTreeModule
依赖模块FormsModule
NgbModule
属性/事件类型说明
entityName属性String实体名
treeTitle属性String树标题
idField属性Stringid字段名
valueField属性String显示字段名
parentField属性String父节点字段名
condition属性ConditionExpr查询条件
nodeSelectEvent事件节点选中事件,返回节点数据

selectedNodeData: {{selectedNodeData.id}} {{selectedNodeData.value}}

图片预览组件 app-image-view

根据图片地址展示图片，并提供点击图片查看大图的功能。

使用标签app-image-view
导入模块ImageViewModule
依赖模块
属性/事件类型说明
appImageView属性String图片地址（默认情况下，为图片相对否台runtime目录的地址）。
uri属性String图片服务器地址，默认为后台api地址，可自行修改，修改后图片的实际地址为：uri+appImageView属性。
imgStyle属性String图片样式，默认样式是宽高48px
