配置项目开发环境

  1. 安装JDK、Idea、WebStorm以及Chrome浏览器（公司电脑都已经预安装）；
  2. 安装NodeJS，从公司内网共享文件夹（file://server.jr.com/ftp/软件/开发环境/）中下载 node-v10.16.3-x64.msi 文件以进行安装；
  3. 使用 Webstorm 从公司SVN仓库检出前端代码，代码地址为：http://svn.jr.com/svn/h5web/projectMgr-www （如果没有SVN账号，请先联系部门负责人以创建SVN账号，并赋予相应权限）；
  4. 使用 Idea 从公司SVN仓库检出后端代码，代码地址为：http://svn.jr.com/svn/fadp13/branches/projectMgr ；
  5. 建议禁用计算机上安装的第三方输入法的快捷键，禁用QQ(除截屏外)的快捷键，如果还有其他软件可能占用开发快捷键请一并禁用。

运行你的代码

  1. 使用Idea打开后端项目，点击Idea右上角的调试按钮启动后端平台，等待启动完成；
  2. 使用WebStorm打开前端项目；
  3. 在项目目录内运行 npm install；
  4. 在WebStorm中点击左下角npm > start，启动angular，等待启动完成；
  5. 在WebStorm中点击右上角调试按钮，启动浏览器调试。

查看前端页面

  1. 浏览器自动打开页面或者在浏览器输入localhost:4200，进入前端主页面；
  2. 如果在浏览器显示出如上图的前端主页面，恭喜你，前后端环境配置成功，现在你可以使用CGT工具生成你的代码了！
